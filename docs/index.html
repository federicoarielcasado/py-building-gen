Descripción del proyecto

py-building-gen es una aplicación de escritorio en PyQt6 que automatiza la generación de scripts Dynamo para Autodesk Revit 2027, permitiendo modelar un edificio residencial multifamiliar completo a partir de parámetros ingresados por el usuario.

El sistema produce 10 scripts .dyn en formato JSON que, ejecutados en orden dentro de Revit, construyen automáticamente la arquitectura, estructura, instalaciones MEP, vistas de documentación y láminas con title block IRAM A3. El módulo de cómputo genera el presupuesto completo con análisis de precios unitarios, exportable a Excel y PDF.

Características principales

  • Interfaz PyQt6 con 5 tabs paramétricos y previsualización en tiempo real (planta + alzado)
  • Predimensionado estructural automático según CIRSOC 201-2005 y CIRSOC 101
  • 10 scripts Dynamo (CPython 3) para Revit 2027: niveles, muros, losas, estructura, aberturas, MEP, vistas y sheets
  • Cómputo métrico por rubro (62 ítems en 15 rubros) con análisis de precios unitarios
  • Precios de referencia: UOCRA Zona A 2026 + ICC INDEC + Sismat
  • Exportación a Excel (hoja por rubro + curva de inversión) y PDF
  • Serialización del proyecto (.pbg) para guardar y reutilizar configuraciones
  • 49 tests automatizados (pytest)

Pipeline de generación

1
Ingreso de parámetros — UI PyQt6 con validación en tiempo real, previsualización de planta y alzado esquemáticos.
2
Predimensionado estructural — core/predimensionado.py calcula secciones de losas, vigas, columnas y zapatas según CIRSOC 201-2005.
3
Generación de scripts Dynamo — 10 archivos .dyn (JSON) con nodos Python (CPython 3) y Code Blocks parametrizados con los valores del proyecto.
4
Ejecución en Revit 2027 — Scripts corridos en orden: niveles → muros → losas → estructura → aberturas → circulación → MEP → vistas → sheets → schedules.
5
Cómputo y presupuesto — core/computo/ genera mediciones, aplica precios UOCRA + INDEC + Sismat y exporta a Excel (con curva de inversión) y PDF.
6
Clash Detection — Exportación a Navisworks Manage 2027 vía NWC para detección de interferencias MEP-estructura.
7
Portfolio — Láminas A3 con title block IRAM 4505 exportadas desde Revit, renders desde Blender, documentación completa por capítulo.
